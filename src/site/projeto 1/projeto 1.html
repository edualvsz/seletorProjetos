Conteudos:



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo de páginas web. Ele introduz novos elementos e funcionalidades que facilitam o desenvolvimento de sites modernos, com foco em interatividade, multimídia e compatibilidade com dispositivos móveis

CSS3 é a terceira versão da linguagem Cascading Style Sheets (CSS), usada para estilizar páginas web, definindo como elementos HTML são exibidos no navegador. Ele introduziu novas funcionalidades, como media queries para layouts responsivos, bordas arredondadas, sombras, gradientes, transformações 2D/3D e animações, além de módulos que facilitam a organização do código.

O que é a propriedade Display em CSS.


Um display (ou mostrador, em português) é um dispositivo para a apresentação de informação, de modo visual e/ou táctil, adquirida, armazenada ou transmitida sob várias formas. Quando a informação de entrada é fornecida como um sinal elétrico, o display é chamado de "display (ou "painel") eletrônico".

Dispaly Caracteristicas
Block Elementos HTML (Linguagem de marcação de hipertexto) historicamente foram categorizados como "nível de bloco" ou elementos "em linha". Um elemento em nível de bloco ocupa todo o espaço de seu elemento pai (container), criando assim um "bloco"
Inline "Inline" refere-se a um tipo de elemento que não começa em uma nova linha e ocupa apenas o espaço necessário para seu conteúdo
Inline-Block Elementos com display: inline-blocksão colocados na mesma linha que outros elementos inline ou inline-block, semelhantes aos inlineelementos . Eles não forçam uma quebra de linha antes ou depois deles.
Flex A propriedade flex é uma propriedade abreviada para: A propriedade flex define o comprimento flexível em itens flexíveis. Nota: Se o elemento não for um item flexível, a propriedade flex não terá efeito.
